🧰 How PySorta Works

It's like organizing your closet by color, season, or how often you wear something — except with code.


1. Ask user for a folder path
2. Ask how they want to sort: by type, date, or size
3. Scan all files in that folder
4. Create subfolders like "Images", "2025-06", or "Large"
5. Move files into their matching category folder
